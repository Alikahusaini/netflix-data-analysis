# Netflix Data Analysis

Data analysis project using Python, Pandas, NumPy, and Matplotlib to explore Netflix content trends, genres, release patterns, and audience distribution.

## Key Analysis
- Movies vs TV Shows distribution
- Top countries producing Netflix movies
- Netflix release trends over time
- Adult vs Non-Adult content comparison
- Genre analysis
- Country-wise content analysis

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
